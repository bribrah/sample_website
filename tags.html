Important HTML Tags

This is designed as a quick reference for some of the common tags covered in Learn Enough HMTL to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags

The tables below don't include all HTML tags, but they do list many of the most important ones.

\
Tag Name Purpose Example Purpose
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong makes strong(bold) text <strong>suh dude</strong> suh dude
a anchor makes a hyperlink go to<a href="http://learnenough.com/">learnenough.com</a> go to learnenough.com
li list adds a list element to a list block <li>tde cow goes moo</li>
  • the cow goes moo
    		•
    ol Ordered List Makes an ordered list block <ol><li>the cow goes moo</li><li>the duck goes quack</li></ol>
    1. The cow goes moo
    2. The duck goes quack
    ul unordered list makes an unordered list block <ul><li>This is an unordered list</li><li>It uses bullet points</li></ul>
    • This is an unordered list
    • It uses bullet points
    img image makes an image <img src="images/storm_trooper_tagged.jpg" alt="Tagged Storm Trooper">
    h1–h6 headings include a heading (levels 1–6)
    p paragraph indlude a paragraph of text
    table table include a table
    tr table row include a row of data
    th table header make a table headr
    td table data include a table data cell
